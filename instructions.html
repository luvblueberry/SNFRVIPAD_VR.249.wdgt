Instructions for sharing KeyShotVR

Below is your KeyShotVR. To view the KeyShotVR by itself open the SNFRVIPAD_VR.249.html file in the SNFRVIPAD_VR.249 folder.

 

Uploading to your web site:

Upload the SNFRVIPAD_VR.249 folder and SNFRVIPAD_VR.249.html to your webserver. Make sure when uploading the folder
and html file that your html file is located in the same directory as the "files" folder inside SNFRVIPAD_VR.249.
Copy/Paste the code below to your webpage.

[Code]: 

<div style="max-width: 2048px; max-height: 1536px;">
  <div style="left: 0px; width: 100%; height: 0px; position: relative; padding-bottom: 75%; overflow: hidden;">
    <iframe src="SNFRVIPAD_VR.249.html"
                 allowfullscreen
                 style="position: absolute; top: 0px; left: 0px; height: 100%; width: 1px; min-width: 100%; *width: 100%;"
                 frameborder="0"
                 scrolling="no">
    </iframe>
  </div>
</div>

If you are sharing via email. You will need to send the entire zipped SNFRVIPAD_VR.249 folder to the end recipient.
